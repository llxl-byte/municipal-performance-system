🧪 测试步骤

  1. 启动服务：确保前端(3000)和后端(3001)服务正在运行
  2. 访问页面：打开 http://localhost:3000/projects
  3. 测试搜索：在项目名称搜索框中输入关键词
  4. 测试分页：切换不同页码和每页显示条数
  5. 测试工具栏：尝试刷新、密度调整、全屏等功能
  6. 测试响应式：调整浏览器窗口大小查看适配效果
